# 평양냉면 용어 사전 및 가이드 데이터 수집 프롬프트

## 프롬프트 4: 전문 용어 사전 수집

```
평양냉면 매니아들이 사용하는 전문 용어를 조사하여 JSON으로 정리해주세요.

### 카테고리별 수집 대상

1. **육수 관련 (broth)** - 최소 15개
   - 맛 표현: 슴슴하다, 심심하다, 깊다, 텁텁하다, 맑다, 진하다
   - 상태 표현: 살얼음, 차갑다, 시원하다
   - 재료 관련: 육향, 동치미, 꿩, 사골

2. **면 관련 (noodle)** - 최소 15개
   - 식감: 툭툭 끊기다, 뚝뚝, 쫄깃, 꼬들꼬들, 수더분하다
   - 품질: 구수하다, 메밀향, 목넘김
   - 상태: 불다, 퍼지다

3. **주문 용어 (ordering)** - 최소 10개
   - 을밀대 특수: 거냉, 양마니
   - 일반: 사리, 곱빼기, 면수
   - 숨은 메뉴: 비빔+물, 반반

4. **문화 용어 (culture)** - 최소 10개
   - 코스: 선육후면, 선주후면
   - 메뉴: 어복쟁반, 편육
   - 현상: 면스플레인, 줄서기 문화

5. **평가 용어 (evaluation)** - 최소 10개
   - 긍정: 정통, 본토맛, 밸런스
   - 부정: 상업화, 퓨전
   - 논쟁: 정통성, 계승

### 각 용어별 수집 항목
- 용어 (한글)

- 긍정/부정 뉘앙스 (해당 시)
- 어원이나 유래 (있을 경우)
